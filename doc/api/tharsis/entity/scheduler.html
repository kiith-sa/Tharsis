ProcessInfo

Information about a process for scheduling purposes.

Classes

Scheduler

Determines which Processes should run in which threads, as well as the thread count.

SchedulingAlgorithm

Base class for scheduling algorithms. Can be derived to implement custom algorithms.

DumbScheduling

Extremely basic/dumb scheduling algorithm.

RandomBacktrackScheduling

Randomized backtracking that decides randomly on which branch of the call tree to take, but covers all branches that it didn't backtrack from.

PlainBacktrackScheduling

Plain, old school, extremely slow backtracking.

LPTScheduling

Longest processing time scheduling algorithm. Fast and gets decent results.

TimeEstimator

Base class for time estimators.

SimpleTimeEstimator

Simple time estimator that expects a process to take the same time it took the last frame.

StepTimeEstimator

Time estimator that quickly increases estimated time at a spike and gradually decreases it as the measured time gets lower.
