public bool loadMultiComponent(
    ref const(ComponentTypeInfo) componentType, 
    ref Source sequence, 
    ref EntityPrototype prototype, 
    void delegate(string) nothrow logError) nothrow

Load multicomponents of type componentType from sequence to prototype.

Parameters

componentType

Type of components to load. Must be a MultiComponent type.

sequence

Source storing a sequence of components.

prototype

Prototype to load the components into. Must not be locked yet.

logError

A delegate to log any loading errors with.

Returns

true if all components successfully loaded, false if a) there are zero components (empty sequence), b) failed to load a component, or c) there are more components than the max number of components of type componentType per entity (this limit is specified in the component struct).
