public void requestLoad(const Handle handle) @safe nothrow

Request the resource with specified handle to be loaded by the manager.

ResourceManager will try to load the resource asynchronously. If the resource is already loaded, requestLoad() will do nothing.

There is no way to force a resource to be loaded immediately; the resource may or may not be loaded by the next frame; it may even fail to load.
