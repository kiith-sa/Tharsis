public abstract class MallocResourceManager(Resource) : ResourceManager!Resource

Base class for resource managers storing resources in manually allocated memory.

Resources created during a frame are temporarily stored in a mutex-protected array, loaded when ResourceManager.update() is called and moved into permanent, immutable storage (consisting of manually allocated pages that are never moved).

Resources are loaded by a delegate passed to MallocResourceManager constructor.
