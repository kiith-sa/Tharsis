public EntityID addEntity(ref immutable(EntityPrototype) prototype) @trusted nothrow

Add a new entity, using components from specified prototype.

The new entity will be added at the beginning of the next frame.

Parameters

prototype

Prototype of the entity to add. Usually, the prototype will have to be cast to immutable before passed. Must exist without changes until the beginning of the next update. Can be safely deleted afterwards.

Returns

ID of the new entity on success. A null ID if more than Policy.maxNewEntitiesPerFrame new entities were added during one frame.
