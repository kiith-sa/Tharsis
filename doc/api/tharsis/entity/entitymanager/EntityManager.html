public class EntityManager(Policy)

The central, "World" subsystem of Tharsis.

EntityManager fullfills multiple roles:

  • Registers Processes and resource managers.
  • Executes processes.
  • Manages game state (past and future entities and components).

Parameters

Policy

A struct with enum members specifying various compile-time parameters and hints. See tharsis.entity.entitypolicy for an example.
