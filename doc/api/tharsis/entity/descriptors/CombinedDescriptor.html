public struct CombinedDescriptor(Resource)

A resource descriptor that can store either a filename to load a resource from or a Source to load the resource directly.

Used when a resource may be defined directly in a subnode of a source (e.g. YAML) file, but we also want to be able to define it in a separate file and use its filename.

Parameters

Resource

Resource type described by the descriptor. Templating by resource type prevents assignments between descriptors of different resource types.
