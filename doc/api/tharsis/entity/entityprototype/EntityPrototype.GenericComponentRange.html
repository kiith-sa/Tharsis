public struct GenericComponentRange(Flag!"isConst" isConst)

A range to iterate over components in an EntityPrototype.

If the isConst parameter is true, the range can only read components. Otherwise the components can be modified.

The components are iterated as RawComponents, i.e. raw byte data with type ID. Usually they will need to be cast to an actual component type to be useful.
