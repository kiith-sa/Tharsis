    Struct ComponentBuffer

    A buffer storing all (either past or future) components of one component

    type.

    The components are stored as plain bytes. Components are written by

    accessing unused memory with unusedSpace(), writing the components, and

    "comitting" them with commitComponents(). If there's not enough space for

    new components, the buffer must be reallocated with

    reserveComponentSpace().

    Fields

    Name Type Description
    allocatedComponents_ size_t The size of storage_ in components.
    committedComponents_ size_t The number of component committed (written) so far.
    componentSize_ size_t Size (in bytes) of a single component in this buffer.
    componentTypeID_ ushort ID of the type of stored components.
    enabled_ bool Is this ComponentBuffer enabled?
    storage_ MallocArray!ubyte The buffer itself. Stores components without any padding.

    Properties

    Name Type Description
    allocatedSize [get] size_t Get the allocated space in components.
    uncommittedSize [get] size_t Get the size of uncommitted (free) space in components.
