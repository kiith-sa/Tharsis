    Struct PartiallyMutablePagedBuffer

    A paged array capable of 'building' and storing immutable data.

    To write to the buffer, the getBytes() method can be used to access a slice

    of the buffer. After writing to this slice, it must be 'locked' by the

    lockBytes() method; this marks the area pointed to by the slice immutable.

    Any further getBytes() calls will access area after the locked data,

    allocating new pages as needed.

    Fields

    Name Type Description
    base_ PagedArrayBase Handles page allocation.
    bytesAwaitingLock_ uint Size of the slice returned by the last call to getBytes().
    immutableSize_ uint The number of immutable bytes in the buffer.

    Methods

    Name Description
    clear Clear the buffer, zeroing out all data (but not deallocating).
    getBytes Gets a slice of all remaining bytes in the current page.
    getBytes Gets a slice of at least specified size.
    lockBytes Lock (mark as immutable) a slice returned by getBytes().
