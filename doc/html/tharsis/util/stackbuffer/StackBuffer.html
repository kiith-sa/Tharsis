    Struct StackBuffer

    An array that uses stack storage (avoiding allocation overhead) if its

    length is short and heap (manual allocation) if its length is long.

    Constructors

    Name Description
    this Disable any slice operations that may assume a GC-allocated array.
    this Construct a StackBuffer of specified size filled with T.init .

    Fields

    Name Type Description
    self_ T[] The buffer itself from the user's point of view.
    stackBuffer_ T[maxStackSize] The stack storage used if the size of the array is <= maxStackSize.
