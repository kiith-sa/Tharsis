    Struct EntityPrototype.GenericComponentRange

    A range to iterate over components in an EntityPrototype.

    If the isConst parameter is true, the range can only read components. Otherwise

    the components can be modified.

    The components are iterated as RawComponents, i.e. raw byte data with type ID.

    Usually they will need to be cast to an actual component type to be useful.

    componentIndex_ size_t Index of the current component in the prototype.
    componentOffset_ size_t Offset, in bytes, where the current component in Prototype.components_
    componentTypeInfo_ const(ComponentTypeInfo)[] Type information about all component types.
    front_ RawComponent Current element of the range (type ID + slice with component data).
    prototype_ EntityPrototype.GenericComponentRange.Prototype* The entity prototype we're iterating over.
